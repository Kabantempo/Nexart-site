## 5. VOS DROITS : COMMENT LES EXERCER ?

### 5.1 Droit d'accès

**Qu'est-ce que c'est** : Demander quelles données nous avons sur vous.

**Comment l'exercer** :
- Email: contact@nexart.fr : "Je demande accès à mes données"
- Ou: Bouton "Télécharger mes données" (bientôt dans settings)
- Délai : 30 jours max
- Format : JSON/CSV téléchargeable

### 5.2 Droit de rectification

**Qu'est-ce que c'est** : Corriger données inexactes.

**Comment l'exercer** :
- Directement dans votre profil : modifier nom, localisation, bio
- Automatique = pas délai

### 5.3 Droit à l'oubli (Suppression)

**Qu'est-ce que c'est** : Demander suppression vos données.

**Comment l'exercer** :
- Settings → "Supprimer mon compte"
- Ou email: contact@nexart.fr : "Supprimer mon compte"
- Soft-delete immédiat (30j droit rétractation)
- Hard-delete après 30j
- Contrats conservés 6-11 ans (obligation légale)

### 5.4 Droit à la portabilité

**Qu'est-ce que c'est** : Recevoir données format standard.

**Comment l'exercer** :
- Email: contact@nexart.fr : "Export mes données JSON/CSV"
- Délai : 30 jours

### 5.5 Droit d'opposition

**Qu'est-ce que c'est** : Refuser emails marketing.

**Comment l'exercer** :
- Clic "Unsubscribe" email marketing
- Ou Settings → décocher "Emails marketing"
- Emails transactionnels continuer

## 7. COOKIES & TECHNOLOGIES SIMILAIRES

### 7.1 Qu'est-ce qu'un cookie ?

Cookie = petit fichier stocké ordinateur. Permet reconnaître vous lors prochaine visite.

**Cookies techniques** (obligatoire, pas consentement requis) :
- Session ID = rester connecté
- CSRF token = sécurité

**Cookies marketing/analytics** (consentement requis) :
- Google Analytics (GTM-PC469WF9) = qui visite pages

### 7.2 Bandeau consentement

**Vous verrez** : Bandeau "Accepter cookies" quand première visite.

**Vos options** :
- ✅ "Accepter tous" = tous cookies activés
- ✅ "Refuser" = juste cookies techniques (défaut)

**Important** : Default = REFUSER (pas pré-coché).

### 7.3 Comment désactiver cookies

Browser paramètres :
- Chrome : Settings → Privacy → Cookies
- Safari : Preferences → Privacy
- Firefox : Preferences → Privacy
